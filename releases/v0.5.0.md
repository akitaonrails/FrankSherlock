# v0.5.0

### In-app updates
- The app now checks for updates on startup and shows a button in the sidebar when a new version is available. Click to download and install without leaving the app.
- Manual "Check for Updates" button in the Tools section for on-demand checks.
- Uses Tauri's built-in updater with signed update artifacts — the app verifies the signature before applying anything.

### Scan progress fix
- The progress bar and file counter during phase 2 (classification) now update after every file instead of batching every 50. With LLM classification taking a few seconds per file, the old batching made progress look frozen for minutes.

### UI
- The Tools section (Find Duplicates, PDF Passwords, Check for Updates) is now pinned above the Info panel in the sidebar. Content lists (roots, albums, smart folders) scroll independently underneath.

### CI
- Release notes are now loaded from `releases/vX.Y.Z.md` and included in GitHub Releases automatically.
